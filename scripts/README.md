# Scripts - 分析脚本工具

本目录包含各类自动化分析脚本，用于提升问题分析效率。

## 📂 目录说明

- **trace-analyzer/** - Systrace/Perfetto自动化分析脚本
- **log-parser/** - 日志解析工具（ANR、Crash、logcat）
- **anr-analyzer/** - ANR日志AI智能分析工具
- **automation/** - CI/CD集成脚本，自动化测试

## 🎯 使用方式

每个工具目录都包含独立的README和使用说明：

```bash
cd <tool-name>
./run.sh --help
```

## 🛠️ 技术栈

- **Python** - 主要开发语言
- **Shell** - 自动化脚本
- **Pandas** - 数据处理
- **Matplotlib** - 可视化

## ✨ 特点

- ✅ 自动化分析，减少人工工作量
- ✅ 支持批量处理
- ✅ 生成可视化报告
- ✅ 支持CI/CD集成
